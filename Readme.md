# Antes de continuar
Es necesario configurar un archivo .env con los siguientes atributos
´GMAIL_USER=tu_correo
GMAIL_APP_PASSWORD=contraseña_de_apps
JWT_ACCESS_SECRET=contraseña_jwt
JWT_REFRESH_SECRET=otra_contraseña_jwt´

El correo indicado en este archivo .env tambien debe ser cambiado en *Database.js*
